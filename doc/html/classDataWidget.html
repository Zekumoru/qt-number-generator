Detailed Description

The DataWidget abstract class shows the "all", "even", and "odd" values based on an internal logic provided by its concrete classes (subclasses of DataWidget).

By some internal logic provided by its subclasses, it will show the current values of "all", "even", and "odd". However, updateInterface() must be called to update the texts in the widget's interface.
