\doxysubsection{Detailed Description}
The \doxylink{classDataWidget}{Data\+Widget} abstract class shows the "{}all"{}, "{}even"{}, and "{}odd"{} values based on an internal logic provided by its concrete classes (subclasses of \doxylink{classDataWidget}{Data\+Widget}). 

By some internal logic provided by its subclasses, it will show the current values of "{}all"{}, "{}even"{}, and "{}odd"{}. However, \doxylink{classDataWidget_a9bbf5dc66885706e89a3d06a58669e32}{update\+Interface()} must be called to update the texts in the widget\textquotesingle{}s interface.
